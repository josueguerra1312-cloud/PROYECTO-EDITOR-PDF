# PDF Tech Editor v4

Aplicacion Streamlit validada contra tres pares reales de referencia.

## Despliegue

Sube todos los archivos a la raiz del repositorio y configura Streamlit Cloud con:

```text
Main file path: pdf_editor.py
Python: 3.12
```

Despues utiliza **Manage app > Reboot app**.

## Archivos

- `pdf_editor.py`: interfaz y motor completo.
- `requirements.txt`: dependencias.
- `runtime.txt`: version de Python.
- `VALIDACION.md`: casos de regresion verificados.
